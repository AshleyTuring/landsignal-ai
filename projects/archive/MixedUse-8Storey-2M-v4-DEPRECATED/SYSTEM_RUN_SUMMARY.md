## üí° KEY LEARNINGS

### 1. Natural Collaboration > Forced Divergence

**v3.1 Problem:**
```
Force Codex to be "creative" even when regulation angle is strongest
‚Üí Codex creates weak "creative regulation" approach to meet quota
‚Üí Dilutes quality, wastes testing time
```

**v3.2 Solution:**
```
All agents explore regulation angle together
‚Üí All agree lease expiry + EPC is strong combined strategy
‚Üí One high-quality approach, no wasted approaches
```

### 2. Systematic Angles Ensure Coverage

**v3.1 Risk:**
```
Hope that 3 different personas happen to cover all important angles
‚Üí In first v3.1 run, NO precedent approach generated
‚Üí User had to manually request it
```

**v3.2 Solution:**
```
Precedent = mandatory Round 1
Other 4 rounds ensure regulation/structural/behavioral/challenge all considered
‚Üí Comprehensive coverage guaranteed
```

### 3. Challenge Round Prevents Groupthink

**Without Challenge Round:**
```
All agents agree on 5-6 safe, obvious approaches
‚Üí Miss speculative high-upside plays
‚Üí Portfolio too conservative
```

**With Challenge Round:**
```
Forces thinking beyond obvious
‚Üí Time arbitrage (emerging allocations)
‚Üí Strategic land promotion
‚Üí High risk/reward balance
```

### 4. Honest Assessment Improves Efficiency

**v3.1:**
```
"Each agent MUST create 2 approaches" = quota system
‚Üí Agent creates weak 2nd approach to meet requirement
‚Üí Wastes validation and testing time
```

**v3.2:**
```
"Generate 0-2 approaches per round if compelling"
‚Üí Can skip weak rounds
‚Üí Quality over quantity
‚Üí More efficient testing
```
